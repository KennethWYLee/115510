# Artorias Castle 戰鬥系統操作台

GitHub repository name: `115510`

第 115510 組系統已整理為可操作的靜態前端工作台，包含互動任務、狀態管理與本機瀏覽器資料保存。

## 系統能力

- 以物理碰撞體判斷攻擊範圍、命中時機、格擋窗口與傷害結算。
- 透過 HUD 呈現生命、體力、道具、敵人狀態與目前戰鬥節奏。
- 建立序列化存檔與角色資料結構，追蹤探索進度、成就與數值。
- 以模組化架構管理場景、光影與美術資源，維持技術實作與藝術表達一致。
