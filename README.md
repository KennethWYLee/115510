# Artorias Castle 戰鬥系統展示中心

GitHub repository name: `115510`

五專第115510組 Artorias castle 系統手冊 的專業化 GitHub Pages 靜態展示版。

## 展示定位

- 中文化、免登入、全功能展示。
- 根據學生文件的原始構想延伸成更接近正式產品的操作介面。
- 純前端靜態檔案，不連接後端、資料庫、第三方 API 或真實校內服務。
- 使用模擬資料，避免公開學生個資、帳密、內部網址與敏感資訊。

## 檔案

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## GitHub Pages

推送至 GitHub 後可於 Settings > Pages 設定：

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
